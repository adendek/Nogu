# Nogu documentation

This repository holds the Nogu project documentation.

The link to the documentation can be found [here](https://adendek.github.io/Nogu/).

## Editing documentation locally

Follow these steps to edit your documentation locally:

### Serve Documentation Locally

```bash
   uv run mkdocs serve
```

### Open the Documentation
After running the above command, you can open the documentation by clicking the link in the command line output:

Now you can start writing your documentation! Make sure to follow any guidelines or templates provided by your team.
